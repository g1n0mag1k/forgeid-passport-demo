# ForgeID Passport Demo

A neutral human-AI authenticity passport tool. All badges are equal. The value of this work is determined solely by its quality, impact, and results — not by the human/AI ratio. Built in public over 7 days.

## 7-Day Plan

1. **Foundations & Equality Charter** — define the ethos, passport promise, and public build scope.
2. **Roadmap UX & Narrative** — design the day cards, timeline, and launch pad CTAs.
3. **Passport Schema** — map the badge and evidence model.
4. **Proof & Verification Flow** — document proof capture and audit trail.
5. **Public Build Ops** — finalize updates, templates, and cadence.
6. **Genesis Passport** — unlock and mint the Genesis state.
7. **Launch & Retrospective** — publish, share, and capture learnings.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- lucide-react
- framer-motion
- canvas-confetti

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the dashboard.
